# TPI-Gestion-Paises
Trabajo Práctico Integrador - Gestión de Países en Python

Este proyecto consiste en una aplicación de consola desarrollada en Python que permite gestionar información sobre distintos países. El sistema carga los datos desde un archivo CSV y ofrece opciones para agregar, actualizar, buscar, filtrar, ordenar y obtener estadísticas sobre los países registrados.


Tecnologías utilizadas

* Python 3.x
* Librería `csv`


Instrucciones de uso

1. Descargar o clonar el repositorio.
2. Verificar que el archivo `paises.csv` se encuentre en la misma carpeta que el programa.
3. Ejecutar el archivo principal con Python.
4. Seleccionar una opción del menú para utilizar las diferentes funcionalidades.
5. Al finalizar, elegir la opción **Guardar y salir** para almacenar los cambios realizados.


Ejemplo de entrada y salida

### Entrada

```
Seleccione una opción: 3
Ingrese nombre o parte del nombre para buscar: argentina
```

### Salida

```
Se encontraron 1 resultados:
Argentina - Población: 45376763 - Superficie: 2780400 km² - Continente: América
```
